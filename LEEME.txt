MoreRobAccounts 2.0 - Lanzador multi-cuenta de Roblox
======================================================

REQUISITOS
----------
- Windows 10 u 11 (64 bits)
- Roblox instalado (roblox.com/download)
- Microsoft Edge (viene con Windows)

INSTALACION
-----------
Opcion A - Instalador (recomendado):
  Ejecuta MoreRobAccounts-Setup-2.0.0-win64.exe

Opcion B - Portable:
  Descomprime el ZIP y ejecuta MoreRobAccountsUI.exe
  (deja accounts.json en la misma carpeta que el .exe)

NOVEDADES 2.0
-------------
- Familias: agrupa cuentas + Place ID y usa "Lanzar Familia"
- Historial de Place ID
- Grupos por cuenta (Farm, Principal, etc.)
- Comprobar / renovar sesiones caducadas
- Validacion antes de lanzar (cuentas invalidas en rojo)
- Barra de progreso y notificaciones de Windows
- Delay configurable si lanzas mas de 3 cuentas

PRIMER USO
----------
1. Pulsa "Iniciar sesion con Roblox" para cada cuenta
2. Crea familias en "Gestionar" (opcional)
3. Pon el Place ID o elige del historial
4. "Lanzar Familia", "Lanzar todas" o "Seleccionadas"
